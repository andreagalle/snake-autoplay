Running a Python project on GitHub Codespaces

Follow these steps to run the snake-autoplay project on GitHub Codespaces.

Step 1: Open Codespace

  1. Once logged-in on GitHub, navigate to the repo andreagalle/snake-autoplay and click on the Use this template > Open in a codespace button.
  2. This will open a browser-based IDE (essentially vscode) in a new tab.

Step 2: Launch Setup Script

  1. From the integrated CLI, launch the setup script after having sourced the hidden .env file: 
                  source .env
              setup
  2. This will install (first time only) all the dependencies we need, basically pygame and a custom version of pygbag modules.

Step 3: Launch Deploy Script

  1. Navigate to the snake version you want to run, for instance the standard one, and launch the deploy script: 
                  cd std/
              deploy
  2. This basically uses the pygbag module passing the GitHub ${CODESPACE_NAME} environment variable to deploy the game on that domain name (randomly chosen by GitHub itself), under the default port 8000.

Step 4: Ready to Start!

Click on the Ready to start! button and that's it!
